# Django OCR App

A simple Django application that extracts text from images using EasyOCR.

---

## How to Run

1. **Install dependencies**:

```bash
pip install django easyocr pillow

1. **Run the Django server**:

python manage.py runserver
3. **Open the app in your browser**:
http://127.0.0.1:8000/ocr/

Upload an image and view the extracted text
